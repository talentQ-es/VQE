De todas las posibles mol√©culas que podr√≠amos tratar, la m√°s sencilla es la de \(\text{H}_2\): el √°tomo de Hidr√≥geno es el m√°s peque√±o y simple, presenta un √∫nico enlace y solo tiene dos electrones.

Mientras que Qiskit puede generar el Hamiltoniano de la mol√©cula con sus propias funciones, con Qibo necesitamos fabricar una funci√≥n que nos convierta el problema molecular en uno legible por un circuito cu√°ntico.


def qibo_mol_H(molecule):

    # molecule es una estructura molecular de OpenFermion
    # devuelve un Hamiltoniano en t√©rminos de Qibo

    # Obtenemos el Hamiltoniano molecular de nuestra mol√©cula fabricada con OpenFermion
    H = molecule.get_molecular_hamiltonian()

    # Nos aseguramos de que el t√©rmino constante est√° bien definido
    if H.constant is None: H.constant = 0

    # Hacemos la transformaci√≥n de Jordan Wigner
    JW = jordan_wigner(H)
    dim = count_qubits(JW)

    # Inicializamos el operador con el t√©rmino constante
    operator = JW.constant * reduce(np.kron, [m.I]*dim)  

    # Recorremos los t√©rminos del operador de Jordan-Wigner
    for index in JW.terms.keys():
        if index == ():
            continue # Omitimos el t√©rmino constante, ya que hemos inicializado con √©l

        # Definimos una lista auxiliar de operadores identidad del tama√±o del sistema
        lis = [m.I]*dim

        # Reemplazamos en nuestra lista los operadores en la posici√≥n en la que indica el t√©rmino de JW
        for tag in index:
            if tag[1] == 'X': lis[tag[0]] = m.X
            elif tag[1] == 'Y': lis[tag[0]] = m.Y
            elif tag[1] == 'Z': lis[tag[0]] = m.Z

        # Sumamos el t√©rmino JW a nuestro operador matricial
        operator += JW.terms[index] * reduce(np.kron, lis)

    return hamiltonians.Hamiltonian(dim, operator), dim, operator

# Ahora creamos un circuito variacional de ansatz
# En este caso usamos un circuito de puertas RY(theta) y CZ, haciendo los diferentes valores de theta aleatorios

op_H, n, mat = qibo_mol_H(molecule)

nqubits = n
nlayers  = 8

circuit = models.Circuit(nqubits)

for l in range(nlayers):
    circuit.add((gates.RY(q, theta=0) for q in range(nqubits)))
    circuit.add((gates.CZ(q, q+1) for q in range(0, nqubits-1, 2)))
    circuit.add((gates.RY(q, theta=0) for q in range(nqubits)))
    circuit.add((gates.CZ(q, q+1) for q in range(1, nqubits-2, 2)))
    circuit.add(gates.CZ(0, nqubits-1))

Para este caso concreto es importante destacar que el problema que hemos resuelto es la diagonalizaci√≥n del Hamiltoniano de Hartree-Fock de la mol√©cula.

Existen otros modelos de ansatz (como los ans√§tze UCC) que, antes que hacer la diagonalizaci√≥n, operan sobre el circuito introduciendo t√©rminos y consideraciones nuevas, como correcciones relativistas o correlaci√≥n electr√≥nica, y despu√©s obtienen la energ√≠a fundamental, de forma que la resoluci√≥n del problema tiene en cuenta efectos m√°s dif√≠ciles de simular con computadores cl√°sicos.
